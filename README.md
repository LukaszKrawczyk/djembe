# Djembe rhythms

Collection of djembe rhythms.

## Legend

* `B` : bass (full hand)
* `T` : tone (half hand)
* `S` : slap (quarter hand)
* `-` : pause
* `|` : group separator

e.g.: `BBTT|-BT-` (8 ticks in 2 groups)
